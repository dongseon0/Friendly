# Friendly : AI Adaptive Horror Game

## 1. 프로젝트 소개
기존의 반복되는 공포게임은 가라!

기존 게임은 모든 플레이에서 같은 스토리, 같은 엔터테인먼트 요소를 제공하여 반복적인 플레이 이후에는 상대적으로 지루한 플레이 경험을 제공하는 경우가 많다.

우리 프로젝트는 이 단점을 문제점으로 보고 AI 기반의 NPC가 플레이어의 행동 패턴을 학습하여, 플레이할 때마다 새로운 방식으로 긴장감을 조성하고 다양한 공포 시나리오를 제공하려한다. 공포·스릴러를 추구하는 게임 유저들에게 정적이고 반복적인 기존 공포게임의 한계를 극복하고, 매번 다른 심리적 공포 경험을 제공하는 것을 목적으로 한다.

Unity 오픈소스 플러그인 ML-Agents 머신러닝 모델을 활용해 Player가 가장 무서워하는 공포 요소를 파악하여 플레이 도중 공포 액션을 개인 맞춤형으로 제공한다.

---

## 2. 주요 특징
### AI 가상친구 NPC
채팅 시스템에서 플레이어와 상호작용하여 PL의 답변에 따라 유동적인 대화를 진행한다.
채팅 시스템, 스토리 진행 상에서 플레이어의 답변, 게임 플레이에서의 음성 정보(비명, 데시벨 등), 마우스 움직임을 관찰하여 플레이어가 가장 반응성이 높은 공포 요소를 파악해 이를 반영하여 공포 액션을 조정한다.
### 적응형 공포 연출
반복적인 패턴이 아닌, 플레이어 성향에 맞는 맞춤형 공포 제공
채팅/플레이 상황에서 플레이어의 성향을 파악해 더 효과적인 공포 전략 적용
